# Active Agent Rules

The project guidelines have been split up for better organization. Please review and adhere to the following rule files in the `rules/` directory before starting tasks:

- **`rules/blueprint.md`**: Outlines the high-level architecture, core objectives, and UI theme design system.
- **`rules/csv-spec.md`**: Defines how the log data is structured and what columns need parsing.
- **`rules/views-spec.md`**: Contains the logic and requirements for the Raw, Daily, and Monthly view behaviors, including all drill-down toggles.
