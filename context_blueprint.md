# Context Blueprint

**Purpose**: A high-performance CSV log analyzer optimized for parsing custom mobile data reports.

## Core Objectives
1. **Efficiency**: Must handle potentially large CSV logs seamlessly.
2. **Usability**: Provide an intuitive drag-and-drop interface.
3. **Aesthetics**: Deliver a premium, dark-mode prioritized interface with rich micro-interactions and excellent visual hierarchy.

## Key Technologies
- **React (Vite)**: For fast rendering and modern development experience.
- **Tailwind CSS**: For robust, custom styling that enables premium designs.
- **PapaParse**: For extremely fast and reliable in-browser CSV parsing.
- **react-dropzone**: For handling file uploads elegantly.
- **lucide-react**: For clean, modern iconography.
